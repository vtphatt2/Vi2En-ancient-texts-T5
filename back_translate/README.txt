+ wget: 
1. Vô trang này: https://eternallybored.org/misc/wget/ và tải file .exe về
2. Dời file wget.exe vào Windows/System32
3. Vào Environment Variable, double click vào Path
4. Chọn New và copy path dẫn tới wget.exe
5. Reset terminal
